# スクリプト　サンプルコード

AI・高度プログラミングコンソーシアム　講習会

## 概要
講習会用のサンプルスクリプト集です。

### ① 一定時間後に消える床
- 一定時間経過後にFloorオブジェクトを削除する処理

### ② Floor（ジャンプ台）でBallに力を加える
- BallがFloorに接触したときに上方向へ力を加える

### ③ 時間差で動く装置
- 指定時間後にオブジェクトが動き出す処理

## 使い方
1. 各スクリプトをプロジェクトに配置  
2. 対象オブジェクトにスクリプトをアタッチ  
3. 必要に応じてパラメータを調整（時間・力の大きさなど）  
4. 実行して挙動を確認
